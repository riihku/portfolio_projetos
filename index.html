Projetos em Ciência de Dados

Construção de um modelo de detecção de doença cardíaca em pacientes.

Uso de Python e ferramentas de estatística e Machine Learning para criar um modelo de classificação de uma base de pacientes de uma empresa de diagnóstico de doenças cardíacas. O intuito era de obter a maior precisão possível na detecção de doenças cardíacas com o objetivo de tornar o negócio lucrativo dado informações de negócio que tornavam a operação inviável. Com a implementação da solução, a empresa teria um lucro de R$ 750.000 mensais considerando uma média de 500 diagnósticos sendo realizados no mês

As ferramentas utilizadas foram:

  • Git, GitHub
  • Python, Pandas, Matplotlib, Seaborn
  • Jupyter Notebook
  • LightGBM
  • SAIBA MAIS

Construção de um modelo de previsão de 6 semanas de vendas das lojas de uma rede de farmácias, colocado em produção no Telegram.

Construído com uso de Python, o modelo sugerido buscar estimar o valor de vendas de cada loja da rede de farmácias Rossmann. Para esse caso, construí um modelo de regressão com séries temporais, alcançando um erro médio percentual de 9,2% do valor de vendas.

As ferramentas utilizadas foram:

  • Git, GitHub
  • Python, Pandas, Matplotlib, Seaborn, Scipy, Sklearn, Boruta
  • Jupyter Notebook
  • XGBoost
  • SAIBA MAIS
